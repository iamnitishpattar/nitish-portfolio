# Personal Portfolio Website

Welcome to the repository for my personal portfolio website!

## Overview
This is a modern, responsive personal portfolio built to showcase my skills, projects, and contact information. It features a sleek dark mode design, advanced glassmorphism, glowing neon accents, and smooth micro-animations.

**Live Demo:** [Deploying Soon]

## 🛠️ Tech Stack
- **Framework**: React + Vite
- **Styling**: Vanilla CSS (CSS Variables, Flexbox/Grid, Keyframe Animations)
- **Design Elements**: Glassmorphism, Mesh Gradients, SVGs

## ✨ Features
- **Animated Hero Section**: Glowing text and custom futuristic AI architecture diagram.
- **Next-Level Dividers**: Traveling neon laser orbs separating sections.
- **Projects Showcase**: Interactive glassmorphic cards featuring exactly 4 curated projects.
- **Contact Modal**: A minimalist popup displaying social links as clean SVG icons.

## 🚀 Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/iamnitishpattar/nitish-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd nitish-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
